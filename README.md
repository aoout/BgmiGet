# BgmiGet

## How to Install

```
pip install bgmiget
```

## Basic Usage

Search anime by search "commnad".

```
bgmiget search "孤独摇滚"
>孤独摇滚[01]xxxxx  0
>孤独摇滚[02]xxxxx  1
>孤独摇滚[03]xxxxx  2
```

Then, download anime by index.

```
bgmiget download 0
```

Filter the searching results by command "include" and "exclude".

```
bgmiget include "蓝天字幕组"
```

```
bgmiget exclude "繁体"
```
